What is Active Directory and what does it do?

It is used to manage and organise information about network resources, such as users, computers, printers, and other devices.

Why is centralised management a more secure option?

This is because if you are in control you can keep people who are not under your domain are in there you can remove them and you can make sure people using the servers have restrictions so they do not go on websites or games they are not meant to go on or meant to download.

What did you do in terms of the configuration?

We had to create a windows server on a virtual machine that had the admin permissions and set up all the ips then we made a user on the windows 10 virtual machine without permissions using the admin account and we had to add certain ips to the user then set it to a domain so we could set the restrictions.

Here are the numbered steps of how i did this

1. Install windows server and windows 10 virtual machines

2. Startup the windows server vm

3. Set up windows server and create an admin account

4. Set ips and make sure everything is correct

5. Open windows 10 vm* and create a user account

6. Add ips to the user account

7. Set the admin restrictions on windows server vm

8. Set up the domain so the admin can change restrictions on windows server vm

9. Test the permissions on windows 10 vm the user

10. If this was all done correctly you would need the admin password to do things for example download certain things
